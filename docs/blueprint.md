# **App Name**: Umbral

## Core Features:

- Mood Tracking: Users can log their mood and Umbral will track their moods.
- Journal Entry Analysis: The system analyzes journal entries to identify topics and feelings.
- Library Chat: Allows users to chat with books using the 'Cero' AI assistant.
- Habit Suggestion: Suggests new habits based on user goals using AI.
- Habit Tracking: Allows users to track their habits. Habits are stored in Firestore.
- Mood Chart: Displays mood chart that stores mood, habits and achievements.
- AI-driven Engagement: The 'Cero' tool will proactively interact with users based on usage patterns, offering timely support and suggestions to engage users.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to evoke calmness and introspection.
- Background color: Very light grayish-blue (#F0F4F8).
- Accent color: Muted violet (#8E24AA) for highlighting important elements and calls to action.
- Body and headline font: 'Alegreya', a serif font, for an elegant and intellectual feel.
- Use minimalist icons for navigation and habit tracking. Icons should be primarily monochromatic, with the accent color used for active states or key actions.
- Employ a clean and spacious layout. Utilize white space generously to reduce visual clutter and improve readability. Cards with rounded corners and subtle shadows should be used to organize content.
- Incorporate subtle animations for user interactions such as habit completion, mood selection, and chart updates. Animations should be smooth and calming, enhancing the user experience without being distracting.
